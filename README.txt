● 說明

1. 遊戲中有兩台坦克，分別為玩家1與玩家2，其操作方法分別為：
   ├ 玩家1：
      → WASD分別控制前後移動、左右轉向；
      → 空白鍵發射砲彈攻擊。
   └ 玩家2：
      → 方向鍵上下左右分別控制前後移動、左右轉向；
      → Enter發射砲彈攻擊。

2. 此遊戲玩法為單機雙人對戰，彼此的血量皆在畫面上方可看見，若遭到砲彈爆炸攻擊後，則會根據玩家與爆炸點的相對距離做等比例的計算，並造成傷害；另外，砲彈不分敵我，若玩家不小心與自己發射的砲彈所造成的爆炸太過接近的話也有可能會遭受波及。最後，將對手擊殺便可贏得該局分數，若率先取得3分者則成為該場遊戲的獲勝者。

3. 遊戲中Camera的視野會隨著兩台坦克的座標而有所改變，若越接近則視野越小，若相距越遠則反之。
